当时没什么头绪，但是发现下午的时候它的 openrasp 崩了 ：）

于是就可以执行命令了，参考下面文章在 java 反弹 shell 然后 cat /flag 就好了

https://blog.spoock.com/2018/11/25/getshell-bypass-exec/
